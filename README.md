Проект симуляция с управляемым юнитом!
Идея проекта: имитация жизни + возможность вмешаться в самобытную жизнь программы;
Символы юнитов: "@" - Игрок; "&" - Босс; "░" - Трава; "▲" - Камень; "~" - Травоядное; "Ѫ" - Хищник;
